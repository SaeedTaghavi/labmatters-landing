LabMatters — Landing Page (Navy / Blue style)
=============================================

Contents
  index.html            The landing page
  assets/               Logos and illustrations (navy/blue styled)
    logo-wordmark.png         Main logo (for light backgrounds)
    logo-wordmark-light.png   Logo for dark backgrounds (used in footer)
    logo-round.png            Round icon mark
    concept.png               Lab scene illustration
    prism.png                 PRISM solution figure
    migration.png             PRISM workflow figure

How to use
  • Keep index.html and the assets/ folder TOGETHER in the same directory.
    (The page links to images by relative path, e.g. assets/concept.png —
     if they get separated, the images won't show.)
  • Open index.html in any web browser, or upload the whole folder to your host.

Note
  Bootstrap, Font Awesome and Google Fonts load from their public CDNs,
  so an internet connection is needed for full styling.
